overall transduction; transduces 'inSeq' with 'inSeqLen' elements to '*outSeqLen' elements in 'outSeq';

Parameters
fstthe finite-state transducer used for transduction
firstSolOnlydetermines whether only the first solution (usually) or all solutions should be produced (for testing); only the last found solution is returned in 'outSeq';
printSolutionif not NULL, every found solution is displayed using the given function
inSeqthe input sequence
inSeqLenthe input sequence length
Return values
outSeqthe output sequence
outSeqLenthe output sequence length
Parameters
maxOutSeqLenmust provide the maximum length of 'outSeq'
altDescBufmust provide a working array of length 'maxAltDescLen'
maxAltDescLenshould be chosen at least 'maxOutSeqLen' + 1
Return values
nrStepsreturns the overall internal number of iterative steps done
Returns
status of the transduction: PICO_OK, if transduction successful
Note
if 'outSeq' or 'altDesc' are too small to hold a solution, an error occurs and the input is simply transfered to the output (up to maximum possible length)
