void env_spec ( sig_innerobj_t *  sig_inObj)

Prepare Envelope spectrum for inverse FFT

Parameters
sig_inObj: sig PU internal object of the sub-object
Returns
void
Remarks
make phase bilateral -->> angh (FFT size, real)
combine in complex input vector for IFFT F = e**(spet/2+j*ang)
Compute energy -->> E (scalar, real)

Input

  • spect (FFT size, real)
  • ang (half FFT size -1, real)
  • m2 fftsize
    • WNr,WNi (FFT size, real) the tabulated sine and cosine values
    • brev (FFT size, real) the tabulated bit reversal indexes Output
  • Fr, Fi (FFT size, complex) the envelope spectrum
  • E (scalar, real) the energy
