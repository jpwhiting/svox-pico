◆ picopal_mpr_alloc()

void * picopal_mpr_alloc ( picopal_objsize_t  size)

Returns a pointer to a newly allocated chunk of 'size' bytes, aligned to the system page size. Memory allocated by this routine may be protected by calling function picopal_mrp_protect().

◆ picopal_mpr_free()

void picopal_mpr_free ( void **  p)

Releases the chunk of memory pointed to by '*p'. 'p' must be previously allocated by a call to picopal_mpr_alloc().

pico_status_t picopal_mpr_protect ( void *  addr,
picopal_objsize_t  len,
picopal_int16  prot 
)

Specifies the desired protection 'prot' for the memory page(s) containing part or all of the interval [addr, addr+len-1]. If an access is disallowed by the protection given it, the program receives a SIGSEGV.
