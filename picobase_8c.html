picoos_uint8 picobase_get_next_utf8char ( const picoos_uint8 *  utf8s,
const picoos_uint32  utf8slenmax,
picoos_uint32 *  pos,
picobase_utf8char  utf8char 
)

Gets next UTF8 character 'utf8char' from the UTF8 string 'utf8s' starting at position 'pos'

Parameters
utf8s: UTF8 string
utf8slenmax: max length accessible in utf8s
pos: position from where the UTF8 character is checked and copied (set also as output to the position directly following the UTF8 char)
utf8char: zero terminated UTF8 character containing 1 to 4 bytes (output)
Returns
TRUE if okay
FALSE if there is no valid UTF8 char or no more UTF8 char available within utf8len

◆ picobase_utf8_length()

picoos_int32 picobase_utf8_length ( const picoos_uint8 *  utf8str,
const picoos_uint16  maxlen 
)

Determines the number of UTF8 characters contained in the UTF8 string 'utf8str' of maximum length maxlen (in bytes)

Parameters
utf8str: a string encoded in UTF8
maxlen: max length (in bytes) accessible in utf8str
Returns
>=0 : length of the UTF8 string in number of UTF8 characters up to the first '\0' or maxlen
<0 : not starting with a valid UTF8 character
Remarks
strict implementation, not allowing invalid utf8
